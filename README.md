# 海南畅核网络科技有限公司开庭资料工作台

这是一个离线静态网页，用来整理被告方开庭材料、证据目录、事实时间线、答辩提纲和可打印文书模板。

## 使用方式

1. 双击打开 `index.html`。
2. 在页面中填写案件信息、证据目录和庭审提纲。
3. 数据会保存在当前浏览器本机，不会上传服务器。
4. 重要节点建议点击“导出备份”保存 JSON 文件。
5. 点击“打印全套”或各模块的“打印本节”生成纸质材料。

## 开庭前核对

- 提交期限、提交份数、是否需要线上提交，以法院送达的应诉通知书、举证通知书和传票为准。
- 如果只有公司被列为被告，个人股东原则上不是当然的个人被告；如果个人也被列为被告，需要另行准备个人答辩和证据。
- 涉及高持股股东责任、财产混同、人格混同时，建议准备公司账户、账簿、纳税、发票、出资和章程等材料，并让律师核对。
