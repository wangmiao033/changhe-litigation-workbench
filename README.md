# 海南畅核网络科技有限公司开庭资料工作台

这是一个部署在 Vercel 上的开庭资料工作台，用来整理被告方开庭材料、证据目录、事实时间线、答辩提纲和可打印文书模板。

## 使用方式

1. 打开 Vercel 网站。
2. 页面会自动加载已经整理好的资料。
3. 在页面中填写案件信息、证据目录和庭审提纲。
4. 收到法院短信或电子送达通知后，录入“法院电子送达事项”模块；带送达参数的专属链接不要公开转发。
5. 修改内容会自动保存，也可以手动点击“立即保存”。
6. 重要节点建议点击“导出备份”保存 JSON 文件。
7. 点击“打印全套”或各模块的“打印本节”生成纸质材料。

“刷新资料”表示从服务器重新读取一次；正常使用时不用点。

## Vercel 配置

项目需要一个环境变量：

- `BLOB_READ_WRITE_TOKEN`：Vercel Blob Store 的读写 token。

建议在 Vercel Dashboard 的项目设置里添加到 Production、Preview、Development 环境。添加或修改环境变量后，需要重新部署。

## 开庭前核对

- 提交期限、提交份数、是否需要线上提交，以法院送达的应诉通知书、举证通知书和传票为准。
- 如果只有公司被列为被告，个人股东原则上不是当然的个人被告；如果个人也被列为被告，需要另行准备个人答辩和证据。
- 涉及高持股股东责任、财产混同、人格混同时，建议准备公司账户、账簿、纳税、发票、出资和章程等材料，并让律师核对。
